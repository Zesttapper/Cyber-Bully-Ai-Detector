# 🛡️ CyberShield – AI Toxic Content Filter (Chrome Extension)

CyberShield is a modern Chrome extension that detects and handles toxic content in real-time across websites.

It highlights or blurs harmful words and phrases using intelligent filtering logic, providing a safer and cleaner browsing experience.

## 🚀 Features

- 🔍 Real-time toxic word detection

- 🌍 Works across dynamic websites (YouTube, social media, etc.)

- 🎨 Multiple modes:

- Highlight mode

- Blur mode (hover to reveal)

- ⚡ MutationObserver support for live content

## 🧠 Category-based classification:

- Severe

- Hateful

- Bullying

- Profanity

- Slang

- Insults

- 🎛 Clean circular UI settings panel

- 🧩 Lightweight & fast DOM processing

- 🔒 No data collection

## 🖥️ How It Works

- The extension:

  - Scans text nodes in the DOM.

  - Detects toxic keywords.

  - Wraps them in styled elements.

  - Applies visual effects (highlight or blur).

  - Observes dynamic content updates in real time.

## 🛠️ Technologies Used

- JavaScript (Vanilla)

- Chrome Extension Manifest V3

- DOM Traversal & MutationObserver

- Dynamic CSS Injection

- Regex-based word detection

## 📦 Installation (Developer Mode)

- Download or clone this repository.

- Open Google Chrome.

- Go to:

  - [Chrome extensions](chrome://extensions/)

  - Enable Developer Mode (top right).

  - Click Load unpacked.

  - Select the project folder.

  - Done ✅
